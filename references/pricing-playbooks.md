# Pricing Playbooks

Use this reference when the user asks about price, test fee, build fee, revenue share, maintenance, source files, or delivery scope.

## Universal Pricing Frame

Use `阶段 + 档位 + 边界`.

| Part | Purpose |
|---|---|
| 测试/方案费 | verify feasibility, produce basis for real quote |
| 基础版 | run the process with manual assistance |
| 标准版 | recommended reusable workflow or stable delivery |
| 高阶版 | near one-click, batch, systemized, or high stability |
| 不包含项 | prevent scope creep |
| 付款节点 | prevent "done first, pay later" |

Never price only by API tokens or hours. Include discovery, solution design, demo, implementation, revision risk, tool cost, opportunity cost, and first-client uncertainty.

## Lane 1: AI Workflow

Use for chat-record short videos, content automation, batch production, internal AI tools, or semi-automatic workflows.

Suggested structure:

| Tier | Range | Use when |
|---|---:|---|
| 测试阶段 | 1500-3000 元 | 1-2 day sample, feasibility, formal quote basis |
| 基础半自动版 | 6000-9000 元 | process works but still needs manual adjustment |
| 标准工作流版 | 10000-15000 元 | reusable workflow, recommended tier |
| 批量/近一键版 | 20000 元起 | higher automation, stable batch production |

Boundary language:

```text
上面主要是工作流搭建费用，不等于后续无限量生成。后续 API、TTS、AI 图片/视频生成、云渲染、工具订阅、素材授权和人工精修，需要按实际使用或另行约定。
```

## Lane 2: Brand Website

Use for independent sites, landing pages, AI brand visuals, English copy, product pages, launch support, or source handoff.

Suggested structure:

| Tier | Range | Use when |
|---|---:|---|
| 已完成阶段补款 | 6500-12000 元 | brand direction, AI visuals, page structure, front-end prototype |
| 基础上线部署 | 1500-3000 元 | static launch, domain/DNS/hosting checks |
| 轻维护 | 800-1500 元/月 | minor text/image updates |
| 标准维护 | 2000-3500 元/月 | scheduled updates, small design/content work |
| 电商/深度定制 | 15000 元起 | payment, backend, SEO, batch product upload, multi-language |

Boundary language:

```text
这不是单纯套模板或几张 AI 图，而是把品牌方向、视觉素材、页面结构、文案表达和可上线前端整合成一个小型定制项目。上线、长期维护、电商支付、批量上新、SEO 运营、源码/设计源文件和素材授权，需要分开确认。
```

## Lane 3: Content Acquisition

Use for lead research, outreach, content marketing, sales follow-up, or creator/business development workflows.

Suggested structure:

| Tier | Range | Use when |
|---|---:|---|
| 线索研究包 | 1500-5000 元 | target profile, lead list, first-contact angle |
| 首轮外联包 | 3000-8000 元 | personalized messages, sequence, response tracking |
| 标准获客工作流 | 8000-20000 元 | reusable research, scoring, outreach, follow-up loop |
| 长期运营 | monthly retainer | continuous content, outreach, reporting |

Boundary language:

```text
获客工作流可以提升研究、触达和跟进效率，但不承诺固定成交额。成交结果还取决于产品、价格、渠道、客户响应、销售跟进和交付能力。
```

## Revenue Share Rule

Do not reject revenue share automatically. Reframe it.

Recommended position:

```text
分成可以作为后续增长合作，但建议和当前基础建设费用分开。当前阶段覆盖的是已经发生的方案、视觉、页面、工作流和上线准备成本；后续等产生可核算订单后，再谈分成比例、统计口径、结算周期、退款售后和我具体负责的增长动作。
```

If making a concession, trade:

- lower price for smaller scope
- lower upfront fee for stricter milestone
- maintenance included for limited time only

Do not trade lower price for unlimited future responsibility.
