# Marketing Case Library

Use this reference when the user asks whether there are real marketing examples, wants stronger self-promotion proof, needs contest positioning, or wants to turn market cases into a practical sales workflow.

## How To Use These Cases

These are public patterns to borrow from, not credentials to claim.

Use this wording:

```text
这个打法借鉴了 <案例名> 的公开模式：<一句话模式>。我们不照搬行业，而是把它改成你的一人公司售前流程。
```

Do not say the user built, won, invested in, partnered with, or operated these projects unless the user explicitly provides evidence.

## Case Notes

### PitchSense

Public pattern: investor matching, valuation reference, AI outreach email, and one-page material in one flow.

Best use in this skill:

- Convert "I can do AI" into "I can help you find the right target, explain the value, and send a first message."
- Build a first-contact package: target profile, pain point, value proposition, short message, one-page summary.
- Use for B2B outreach, partnership development, agency prospecting, and service packaging.

Sendable framing:

```text
我不建议一开始做通用 CRM。更适合先做一个 PitchSense 式的小闭环：只针对一个窄行业，先筛出最值得接触的对象，再生成一条个性化价值主张和一页纸介绍。
```

### Content Creator Connector

Public pattern: input a company, research the brand, match creators or collaboration targets, then draft personalized collaboration emails.

Best use in this skill:

- Turn acquisition into a repeatable "research, match, message" workflow.
- Use public brand signals, social content, product pages, and visual style as first-message material.
- Adapt the pattern for local brands, SaaS companies, independent e-commerce, restaurants, training providers, or agencies.

Sendable framing:

```text
这里可以借鉴 Content Creator Connector 的模式：先研究品牌，再匹配对象，最后写个性化首封消息。你的版本不一定是找达人，也可以是找潜在客户、渠道方或合作品牌。
```

### Resonance AI

Public pattern: voice agent outreach and feedback collection, with structured results written back to a table.

Best use in this skill:

- Turn vague follow-up into standard questions and structured records.
- Collect post-call objections, budget signals, decision timeline, and next action.
- Adapt to WeChat follow-up, forms, meeting notes, or delivery review instead of only phone calls.

Sendable framing:

```text
这个跟单动作可以借鉴 Resonance AI 的反馈采集思路：不要只问“你考虑得怎么样”，而是固定问预算、决策人、阻碍、下一步时间，并沉淀到台账。
```

### Heydesk

Public pattern: a solo developer built an agentic support platform with knowledge import, AI answer, ticket routing, human handoff, support portal, and chat.

Best use in this skill:

- Show that a one-person operator can package multiple AI capabilities into a customer-facing workflow.
- Use for support automation, FAQ, after-sales, onboarding, and recurring client questions.
- Shift the pitch from "I use RAG" to "I reduce repeated support and follow-up work."

Sendable framing:

```text
Heydesk 的启发不是炫技术栈，而是把知识库、客服、工单和人工接管做成一个客户能试用的流程。你的案例页也应该展示流程，而不是只列工具。
```

### PresentMe

Public pattern: generate professional, brand-consistent presentations from topics, PDFs, reports, or GitHub repositories.

Best use in this skill:

- Package past delivery notes, code, screenshots, and client outcomes into proposals or case pages.
- Help one-person companies respond faster in sales calls.
- Use as the model for "materials-to-proposal" automation.

Sendable framing:

```text
可以借鉴 PresentMe 的方向：把散落的交付资料、截图、文档和仓库整理成提案/PPT/案例页。对谈单来说，这比单纯讲“我能做”更有说服力。
```

### Awwwards / Crazy Creative

Public pattern: personal portfolio as an award-worthy marketing asset.

Best use in this skill:

- Treat the user's own website, portfolio, or BotLearn Skill page as a conversion surface.
- Make proof visible in the first screen: who it helps, what outcome it creates, and what evidence supports it.
- Use for brand website, design, interaction, and creative service positioning.

Sendable framing:

```text
Awwwards 这类案例的重点是：作品集本身就是营销资产。你的 Skill 页面也应该让人一眼看到对象、痛点、产出和可验证证据。
```

### Red Bull Basement / AgriConnect

Public pattern: founder story, real user interviews, AI-assisted business plan, and live pitch.

Best use in this skill:

- Build a contest story from real problem, real user, simple demo, and future growth path.
- Avoid presenting the skill as a prompt collection; present it as a sales operating system for one-person companies.
- Use customer conversations and practical notes as proof.

Sendable framing:

```text
比赛叙事可以借鉴 Red Bull Basement 的结构：真实问题、真实用户、一个能演示的解决方案、以及可扩展的商业路径。你的故事核心不是“我做了一个 Skill”，而是“一人公司怎样更稳定地获客和谈单”。
```

## Output Rule

When adding a case, keep it practical:

```text
可借鉴案例：<案例名>
借鉴点：<一句话>
改成你的场景：<一句话>
今天能做的动作：<一条可执行动作>
可直接发送的话术：<短消息>
```

Avoid long case summaries unless the user explicitly asks for research notes.
